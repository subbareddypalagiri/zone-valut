ℹ️ About ZoneVault

Built for resilience, designed for communities

ZoneVault is a modular, offline-capable web app that consolidates emergency contacts, local services, survival guidance, and real-time alerts for selected zones. It's designed to empower users with reliable, zone-specific information—even without internet access.

  • Version: 1.0.0
  • Founder: WT
  • Tech Stack: HTML, CSS, JS, JSON, Service Workers
  • Usage: Navigate via the top menu. Select a zone to view details. Toggle dark mode with the moon icon. Install for offline use.
